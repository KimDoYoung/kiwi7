# kiwi7 : 키움증권 주식매매 

## 목표
- alpine.js와 Tailwind CSS, DaisyUI를 이용하여 restful로 받은 데이터를 화면에 표시
- 디자인 원칙: 많은 정보를 한 눈에 인식되는 통일된 UI
- 이 폴더에서는 단지 html, javascript, css만으로 코드를 만들고 화면에 정갈하고 아름답게 표현하는데 주력
- javascript의 동작을 구현하는데 집중
- 여러개의 html형식과 css를 테스트해서 가독성 좋은 아름다운 웹페이지를 만들어 본다.

## 지침

- 한국어로 주석을 달것.
- 만들어지는 html에서 javscript는 주로 cdn을 이용할 것

### 정렬 및 포맷팅 원칙
1.  **제목과 내용의 정렬 일치**: 테이블 헤더의 정렬은 본문 데이터의 정렬과 반드시 일치해야 한다.
2.  **문자 정렬**: 기본적으로 **왼쪽 정렬** (예: 종목명, 최근추세). 단, **종목코드**는 **중앙 정렬**.
3.  **숫자 정렬**: 모두 **오른쪽 정렬** (예: 전일대비, 등락율, 가격 등).
4.  **색상**:
    -   상승/수익/매수: **빨간색** (`text-red-600`)
    -   하락/손실/매도: **파란색** (`text-blue-600`)
5.  **숫자 포맷**: 금액 및 수량은 반드시 천 단위 콤마를 포함한다.
